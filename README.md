# Калькулятор стоимости обеда в столовой

## Назначение программы
Программа рассчитывает итоговую стоимость обеда в столовой на основе выбранных блюд, их количества, а также учитывает скидки и наценки.

## Входные данные
- Список выбранных блюд (минимум 3 типа: суп, горячее, напиток, салат, десерт)
- Количество каждого блюда
- Наличие скидки по карте лояльности (%)
- Наличие наценки за обслуживание (%)

## Выходные данные
- Стоимость каждого блюда с учётом количества
- Промежуточная сумма
- Сумма скидки
- Сумма наценки
- Итоговая стоимость обеда

## Функциональные требования
1. Ввод данных пользователем (блюдо → цена → количество)
2. Расчёт базовой стоимости: сумма(цена × количество)
3. Применение скидки (процент от суммы)
4. Применение наценки (процент от суммы после скидки)
5. Вывод детализированного чека
6. Логирование каждого расчёта в файл
7. Сравнение двух вариантов обеда (опционально)
