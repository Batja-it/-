# Математическая модель оптимизации выбора обеда

## Постановка задачи
Необходимо подобрать набор блюд из доступного меню, стоимость которого максимально близка к заданному бюджету пользователя.

## Переменные
Пусть:
- `n` — количество блюд в меню
- `x_i ∈ {0,1}` — бинарная переменная, указывающая, выбрано ли блюдо i
- `p_i` — цена блюда i
- `c_i ∈ {1,2,3,4,5}` — количество порций блюда i
- `B` — бюджет пользователя
- `d` — флаг наличия скидочной карты (0 или 1)
- `h` — час посещения
